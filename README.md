# Python图书管理系统 python1237

## 项目概述

本项目是基于Python语言的图书管理系统，采用Flask框架进行Web开发，以MySQL作为数据库存储，利用HTML、CSS和JavaScript实现前端界面设计和交互。

## 技术栈

- 后端：Python + Flask
- 数据库：MySQL
- 前端：HTML + CSS + JavaScript

## 功能模块

- 用户模块
  - 登录
  - 注册
- 图书信息管理
  - 添加图书
  - 修改图书信息
  - 查询图书
  - 删除图书
- 借阅记录管理
  - 查询图书借阅记录
- 读者信息管理
  - 查询读者信息

## 系统角色

- 系统管理员：负责图书和读者信息的管理
- 普通用户：可以进行图书查询和借阅

## 运行环境

- Python环境
- Flask框架
- MySQL数据库

## 部署步骤

1. 环境配置：安装Python、Flask、MySQL及相关依赖
2. 初始化数据库：创建数据库和表结构
3. 部署应用：运行Flask应用
